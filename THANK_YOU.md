# Acknowledgments

ScalAble would not be possible without the open-source ecosystem and tools below.

## Core frameworks

- **[Flask](https://flask.palletsprojects.com/)** — Web application framework
- **[Flask-Session](https://flask-session.readthedocs.io/)** & **[Redis](https://redis.io/)** / **[KeyDB](https://docs.keydb.dev/)** — Session storage
- **[Flask-SocketIO](https://flask-socketio.readthedocs.io/)** — Real-time notebook kernel communication
- **[LangChain](https://www.langchain.com/)** — LLM orchestration and multi-provider support

## Data and analytics

- **[pandas](https://pandas.pydata.org/)**, **[DuckDB](https://duckdb.org/)**, **[PyArrow](https://arrow.apache.org/docs/python/)** — Query materialization and analytics
- **[Plotly](https://plotly.com/python/)** — Interactive visualizations in the notebook kernel
- **[Jupyter](https://jupyter.org/)** (`jupyter-client`) — Sandboxed Python execution

## Database drivers

- **[psycopg2](https://www.psycopg.org/)** / **[ADBC](https://arrow.apache.org/docs/python/api/adbc.html)** — PostgreSQL connectivity
- Connectors and drivers for MySQL, MongoDB, SQL Server, BigQuery, and Snowflake (as integrated)

## LLM providers

- **[Ollama](https://ollama.com/)** — Local model hosting
- **[OpenAI](https://openai.com/)**, **[Anthropic](https://www.anthropic.com/)**, **[Google Gemini](https://ai.google.dev/)** — Cloud model APIs
- **[LM Studio](https://lmstudio.ai/)** — Local OpenAI-compatible server

## Frontend

- **[Tailwind CSS](https://tailwindcss.com/)**, **[Alpine.js](https://alpinejs.dev/)**, **[HTMX](https://htmx.org/)** — UI styling and interactivity
- **[MathJax](https://www.mathjax.org/)** — Mathematical notation rendering

## Foundations & Organizations

- **[Apache Software Foundation](https://www.apache.org/)** — For Apache Arrow, ECharts, and championing the open-source data ecosystem

## Inspiration

- Text-to-SQL research and datasets (e.g. Spider-style benchmarks) that shaped the NL→SQL approach
- The database community’s work on accessible analytics interfaces

Thank you to all maintainers and contributors of these projects.
